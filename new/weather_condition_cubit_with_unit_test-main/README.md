#### Features
- BLoC (Cubit)
- MVVM & Clean Architecture
- Equatable
- Dio - City List & Weather Condition (Two different requests)
- Location informations (Geocoding & Geolocator)
- Constants & Extensions & Enums
- Responsive Design
